## 🛠️ Cara Menjalankan Kode

Untuk menjalankan program JavaScript di repositori ini secara lokal, pastikan Anda telah menginstal [Node.js](https://nodejs.org/).

1. Buka terminal atau command prompt.
2. Arahkan ke direktori berkas yang ingin dijalankan, misalnya:
   ```bash
   cd hari_pertama
   ```
3. Jalankan berkas menggunakan perintah `node`:
   ```bash
   node nama_file.js
   ```
   *Contoh:*
   ```bash
   node programsaldo.js
   ```
